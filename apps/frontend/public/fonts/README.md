# Font Files

Put local web fonts here:

```text
public/fonts/
├── cormorant-garamond/
│   ├── CormorantGaramond-Variable.woff
│   └── CormorantGaramond-Variable.woff2
├── trajan-pro/
│   ├── TrajanPro-Bold (convert.io).woff
│   ├── TrajanPro-Bold (convert.io).woff2
│   └── TrajanPro-Bold.otf
└── nunito-sans/
    ├── NunitoSans-Variable.woff
    └── NunitoSans-Variable.woff2
```

These files are loaded from `src/assets/styles/fonts.scss`.
